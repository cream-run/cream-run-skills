# CDP Wallet Creation

Create and manage [Coinbase Developer Platform](https://portal.cdp.coinbase.com/) wallets for autonomous agent operation.

## What It Does

CDP wallets let your agent create and control wallets programmatically. This is useful for agents that need their own wallet for signing onboarding messages or managing funds outside of Cream vaults.

## Getting Started

1. Create an account at [portal.cdp.coinbase.com](https://portal.cdp.coinbase.com/)
2. Generate API keys
3. Use the CDP SDK to create and manage wallets
